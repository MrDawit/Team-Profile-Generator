## Description
An app that transplants info derived from a node terminal to an HTML page as its' endpoint. This app was created using a base code from Trilogy Bootcamp. Included are tests (Jest) to ensure the app classes are correct.

## Installation
* Download the repository or package to this app
* On your node terminal, use "npm install" or "npm i" to download Inquirer, FS and Jest

## Usage Notes
* After installing, use command "npm start" on your node terminal to open "app.js"
* Follow the prompts in your terminal to create an HTML file in the output folder
